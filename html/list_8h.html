bool list_insert ( list_t *  list,
int  index,
void *  address 
)

Inserts a new element at a given index.

Valid indexes are [0..size].

Example:

list_t *l = list_new(); // l == [] list_insert(l, 0, 42); // l == [42] list_insert(l, 0, 43); // l == [43, 42] list_insert(l, 1, 44); // l == [43, 44, 42] list_insert(l, 5, 45); // l == [43, 44, 42]

The last case fails (and returns false) because size is 3, meaning 5 is not a valid index.

Note that insert at index 0 is the same as prepend and insert index size is the same as append.

Negative indexes should be supported:

list_insert(l, -1, 45); // l == [43, 44, 42, 45]

A positive index can be calculated from a negative like this: pos_i = size + 1 + neg_i.
